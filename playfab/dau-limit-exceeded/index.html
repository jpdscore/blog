DAU Limit Exceeded について

Last Update: 2025-07-24 feedback 共有

こんにちは、Japan Developer Support Core チームの近澤です。
PlayFab を利用しているタイトルにおいて、API 呼び出し時に以下のようなエラーが発生することがあります。

このエラーそのものは、DAU（Daily Active Users）制限を超過したことを意味します。
しかし、実際にはユーザーが過剰に増加したといった状態ではない場合でも本エラーが突然発生する場合があります。

エラーの原因と対処方法

最も一般的な原因は、 支払いの未処理や失敗によるアカウントの一時停止です。

PlayFab は、支払いがしばらく滞った場合に API 利用を制限することがあります。
例えば、クレジットカードの期限切れなどで支払いが停止した場合に発生します。
このエラーが発生するようになると、全てのユーザーが PlayFab へログインできない状態となります。
本番環境を運用いただいている場合は、非常にインパクトの大きい障害となってしまいますため、費用の支払い状況については十分ご注意ください。

適切に支払いが完了していない場合は、事前に請求先のメールアドレスに ar@playfab.com / plfwaccr@microsoft.com からメールが送られます。
このメールが登録のメールアドレスに届いているか確認いただければ、支払いが滞っているかどうか判断できます。

支払い設定を正しく変更した後は、タイトルの復旧手続きをしていただく必要があります。
通知元のメールアドレスへご返信いただき、タイトルの有効化を依頼ください。

相談先について

何か支払いや課金に関して相談されたいことがあれば、下記の窓口を用意しています。
有効化手続きなどの際は下記よりご連絡ください。
