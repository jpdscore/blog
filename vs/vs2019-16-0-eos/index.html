Visual Studio 2019 version 16.0 サポート終了のご案内

Last Update: 2021-01-13 feedback 共有

こんにちは、Japan Developer Support Core チームの松井です。
今回は Visual Studio 2019 version 16.0 のサポート終了についてご案内します。

Visual Studio 2019 version 16.0 のサポート終了について

Visual Studio 2019 version 16.0 (リリース初期のバージョン) は、以下のドキュメントに記載されている日のとおり 2021 年 1 月 12 日 (日本時間 2021 年 1 月 13 日) でサポートを終了します。

https://docs.microsoft.com/ja-jp/lifecycle/products/visual-studio-2019

サポートの終了後、version 16.0 についてはセキュリティ修正を含め更新プログラムの提供が行われませんのでご注意ください。製品を安全にお使いいただくために、最新のマイナー バージョン (本日時点では version 16.8)、もしくはより新しいサービス ベースライン (本日時点では version 16.4 および version 16.7) にアップグレードしていただくようお願いします。

Visual Studio 2019 のサポートライフサイクルについて

Visual Studio 2019 は 5 年間のメインストリーム サポートと 5 年間の延長サポートが設定されています。ただし、すべてのマイナー バージョンではなく、最新のマイナー バージョンおよびサービス ベースラインとして指定されたマイナー バージョンのみがサポートされるバージョンとなります。また、サービス ベースラインとして指定されたマイナー バージョンは、次のサービス ベースラインのリリース後から 1 年間のみサポートが継続され、その後はサポートが終了します。サポートされた安全な状態を維持するために、Visual Studio 2019 のサポート ライフライクルに沿ってアップグレードの計画と実施をお願いします。

Visual Studio のサポート ライフサイクルの詳細については以下のドキュメントをあわせてご確認ください。
https://docs.microsoft.com/ja-jp/visualstudio/releases/2019/servicing
